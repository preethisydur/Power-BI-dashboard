**Sales Performance & Revenue Analytics Dashboard**

**Project Overview**
This project is an interactive Sales Analytics Dashboard built using Power BI.
The objective of this project is to analyze sales performance, identify trends, and generate meaningful business insights from a dataset containing 9000 sales records.
The dashboard provides dynamic visualizations and key performance indicators to support data-driven decision-making.

**Objectives**
Analyze overall sales performance
Compare revenue across regions and products
Identify monthly sales trends and variations
Understand customer payment behavior
Build an interactive and professional dashboard

**Dataset Description**
The dataset contains 9000 records with the following fields:
Order ID
Order Date
Year, Month, Quarter (Extracted from Order Date)
Region
Product
Category
Salesperson
Units Sold
Unit Price
Total Sales
Payment Mode

**Tools & Technologies Used**
Power BI Desktop
Power Query (Data Cleaning & Transformation)
DAX (Data Analysis Expressions)
Microsoft Excel (Dataset Preparation)

**Data Cleaning & Processing**
The following steps were performed:
Verified and corrected data types
Extracted Year, Month Name, Quarter from Order Date
Created Month Number column for proper month sorting
Created DAX measures for KPI calculations
Structured data for optimized visualization

**Key Performance Indicators (KPIs)**
The dashboard includes the following KPIs:
Total Revenue
Total Orders
Total Units Sold
Average Order Value
These KPIs provide a quick overview of business performance.

**Visualizations Used**
Treemap – Revenue contribution by Region
Stacked Bar Chart – Product performance by Region
Line Chart – Monthly Sales Trend
100% Stacked Column Chart – Payment Mode Distribution
Matrix Table – Region & Product level breakdown
Interactive Slicers – Region, Year, Product

**Key Insights**
Identified revenue contribution across different regions
Analyzed product performance distribution
Observed monthly revenue variations
Evaluated customer payment preferences

**Dashboard Features**
Fully interactive filters (Slicers)
Dynamic data updates across visuals
Professional layout and structured design
Exportable and publishable report

**How to Use**
Open the .pbix file in Power BI Desktop
Use slicers to filter by Region, Year, or Product
Explore KPIs and visual insights
Export or publish the dashboard as needed

**Conclusion**
This project demonstrates practical implementation of data visualization and business intelligence techniques using Power BI.
It highlights the ability to transform raw data into actionable insights through interactive dashboards.
